# CCNA Lab Simulation

## Overview
This project demonstrates networking concepts commonly covered in CCNA studies, including network design, subnetting, VLAN segmentation, routing concepts, and troubleshooting.

The goal is to simulate a small enterprise network and document how devices communicate across different departments.

---

## Skills Demonstrated
- IP Addressing
- Subnetting
- VLAN Design
- Network Topology Design
- Router and Switch Concepts
- Network Documentation
- Troubleshooting Methodology

---

## Network Scenario
A company has three departments:

- Administration
- Sales
- IT Support

Each department must be separated into its own VLAN while maintaining connectivity through routing.

---

## Project Structure

- topology/ → Network diagrams
- subnetting/ → IP addressing plans
- vlan-design/ → VLAN configuration documentation
- routing/ → Routing concepts and configurations
- troubleshooting/ → Troubleshooting scenarios
